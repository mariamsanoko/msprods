# Architecture SaaS

- `apps/web` : frontend Next.js pour msprods.fr.
- `apps/api` : backend Express BRAIN_MS pour auth, chat IA, accès et facturation.
- `database` : schéma Supabase et politiques RLS.
- `security` : briques transverses JWT, chiffrement, webhooks et rate limiting.
- `infra` : dossiers de configuration de déploiement.
